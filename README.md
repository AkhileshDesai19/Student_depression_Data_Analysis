# Student Depression Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) on a Student Depression Dataset to identify factors associated with depression among students.

The analysis focuses on academic, financial, lifestyle, and mental health-related factors to uncover meaningful insights that can support student well-being initiatives.

---

## Dataset Information

* Records: 27,901
* Features: 18
* Target Variable: Depression

### Key Features

* Age
* Gender
* CGPA
* Academic Pressure
* Financial Stress
* Study Satisfaction
* Sleep Duration
* Dietary Habits
* Suicidal Thoughts
* Family History of Mental Illness
* Depression

---

## Project Workflow

6. Exploratory Data Analysis

   * Univariate Analysis
   * Bivariate Analysis
   * Multivariate Analysis
7. Insight Generation

---

## Key Findings

* Students aged 21–25 show higher depression levels.
* Higher academic pressure is associated with increased depression.
* Financial stress is one of the strongest contributors to depression.
* Students with suicidal thoughts show significantly higher depression rates.
* Family history of mental illness increases depression likelihood.
* Lower study satisfaction is linked with higher depression.
* Low sleep duration combined with high study/work hours increases depression risk.
